# ROBOTIS Lab Reinforcement Learning

This repository provides tutorials and tools for reinforcement learning and imitation learning with ROBOTIS robots. It enables training policies in simulation and deploying them on real hardware via Sim2Real transfer, facilitating research in robot learning.

## Key Features
*   Tutorials for reinforcement learning and imitation learning.
*   Sim2Real pipeline for deploying trained policies to physical robots.
*   Support for various ROBOTIS robot platforms.

## Tech Stack
*   Python
*   ROS (Robot Operating System)
*   Gazebo / MuJoCo (Simulation)
*   OpenAI Gym / RLlib

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/robotis_lab.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Follow the tutorial notebooks in the `examples/` directory to begin training.
