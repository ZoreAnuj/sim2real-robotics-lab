# robotis_lab

This repository provides tutorials and tools for reinforcement learning and imitation learning using ROBOTIS robots. It enables training policies in simulation and deploying them on real hardware via Sim2Real functionality, facilitating research in robot learning and control.

## Key Features
* Tutorials for reinforcement learning and imitation learning.
* Sim2Real pipeline for policy deployment on physical ROBOTIS robots.
* Support for various ROBOTIS robot platforms.

## Tech Stack
* Python
* ROS (Robot Operating System)
* OpenAI Gym / Gazebo
* PyTorch

## Getting Started
Clone the repository and install dependencies:
```bash
git clone https://github.com/zoreanuj/robotis_lab.git
cd robotis_lab
pip install -r requirements.txt
```
