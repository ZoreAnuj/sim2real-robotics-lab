# Robotis Lab: Reinforcement Learning for Robotics

This repository provides tutorials and tools for reinforcement learning (RL) and imitation learning using ROBOTIS robot platforms. It enables training policies in simulation and deploying them on real hardware via Sim2Real pipelines, facilitating practical robotics research.

## Key Features
*   Tutorials for reinforcement and imitation learning with common robotics tasks.
*   Sim2Real support for transferring trained models from simulation to physical ROBOTIS robots.
*   Pre-configured simulation environments for rapid prototyping and testing.
*   Modular code structure for extending to new robots and tasks.

## Tech Stack
*   Python
*   ROS (Robot Operating System)
*   OpenAI Gym / Gazebo
*   PyTorch

## Getting Started
```bash
git clone https://github.com/zoreanuj/robotis_lab.git
cd robotis_lab
pip install -r requirements.txt
# Follow the setup instructions in /docs for simulation and hardware.
```
