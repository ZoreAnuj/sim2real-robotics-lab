# Sim2Real Robotics Lab

This repository provides tutorials and tools for reinforcement learning and imitation learning using ROBOTIS robots. It enables training policies in simulation and deploying them on real hardware, bridging the gap between simulated and physical robotic control.

## Key Features
* Tutorials for reinforcement learning and imitation learning.
* Sim2Real functionality for policy deployment on real ROBOTIS robots.
* Support for various ROBOTIS robot platforms.

## Tech Stack
* ROS (Robot Operating System)
* Gazebo / MuJoCo (Simulation)
* Python, PyTorch/TensorFlow (Learning frameworks)

## Getting Started
```bash
git clone https://github.com/zoreanuj/sim2real-robotics-lab.git
```
