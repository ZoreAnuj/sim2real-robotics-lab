# robotis_lab

This repository provides tutorials and tools for reinforcement learning and imitation learning using ROBOTIS robots. It enables training policies in simulation and deploying them on real hardware via Sim2Real functionality, facilitating accessible robotics research and development.

## Key Features
*   Tutorials for reinforcement learning and imitation learning.
*   Sim2Real support for policy deployment on physical robots.
*   Pre-configured environments for ROBOTIS platforms.

## Tech Stack
*   Python
*   ROS (Robot Operating System)
*   Gazebo / MuJoCo (Simulation)
*   OpenAI Gym / RLlib

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/robotis_lab.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Follow the setup instructions in the `/tutorials` directory.
