# ROBOTIS Lab: Reinforcement & Imitation Learning Tutorials

This repository provides tutorials for training reinforcement learning and imitation learning agents with ROBOTIS robots. It enables Sim2Real transfer, allowing policies learned in simulation to be deployed on physical hardware.

## Key Features
* Tutorials for reinforcement learning and imitation learning using ROBOTIS robot models.
* Sim2Real functionality for deploying trained policies to real robots.
* Support for various simulation environments and robot platforms.

## Tech Stack
* Python
* ROS (Robot Operating System)
* Reinforcement Learning frameworks (e.g., OpenAI Gym, Stable Baselines3)
* Simulation environments (Gazebo, PyBullet)

## Getting Started
```bash
git clone https://github.com/zoreanuj/robotis_lab.git
cd robotis_lab
pip install -r requirements.txt
# Follow the specific tutorial instructions for your robot and learning task.
```
